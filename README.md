# Wondro Ecommerce
This is a non-functional ecommerce website for my personal portfolio

## Getting Started
To get all the packages.
```bash
npm install
# or
yarn install
# or
pnpm install
```
To launch the app.
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
